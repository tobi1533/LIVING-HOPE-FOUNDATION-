🌿 Living Hope Foundation Website

A modern, responsive website built to showcase the mission, impact, and activities of Living Hope Foundation. This project highlights community outreach efforts, supports engagement, and provides a clean digital presence for the foundation.

---

🚀 Features

- 🎨 Clean and modern design
- 📱 Fully responsive layout (mobile-friendly)
- 🎥 Animated sections on scroll
- 🖼️ Dynamic background image slideshow
- 📸 Image gallery showcasing real outreach work
- 💬 WhatsApp contact button
- 💚 Simple donation call-to-action

---

🛠️ Built With

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (vanilla)

⚙️ How to Use

1. Clone the repository:

git clone https://github.com/your-username/foundation-website.git

2. Open the project folder

3. Run the website:

- Simply open "index.html" in your browser

---

📸 Customization

- Replace images inside the "/images" folder
- Update text content in "index.html"
- Modify colors and animations in the "<style>" section
